## Campaign Description

### Business Context
RIOS LLC operates a VIP List program that provides early access to exclusive opportunities, promotional offers, and special deals. The business serves customers who want to be notified about new opportunities before they are made available to the general public.

### Application/Service
The service is delivered through a web-based opt-in form where users can join the VIP List. Users provide their contact information (name, email, phone number) and explicitly consent to receive SMS messages.

### End Users
End users are individuals who:
- Want early access to exclusive opportunities
- Are interested in promotional offers and special deals
- Have opted in to receive SMS notifications from RIOS LLC
- Are located in the United States (US phone numbers required)

---

## Complete User Workflow

### 1. User Discovery
Users discover the VIP List program through:
- Website marketing
- Referrals
- Direct navigation to the opt-in page

### 2. User Sign-Up Process
1. User visits the public opt-in page (no login required)
2. User sees program description: "Get early access to hidden opportunities before they are made public right to your phone"
3. User views customer care contact information (email and phone)
4. User fills out form with:
   - First Name (required)
   - Last Name (required)
   - Email Address (required)
   - Phone Number (required, US format)
5. User reviews program description section that explains:
   - Types of messages they will receive
   - Message frequency (up to 10 messages per month)
6. User provides explicit consent by checking a required checkbox
   - Checkbox is NOT pre-checked
   - Consent text clearly explains what they're consenting to
   - Includes all required disclosures (rates, frequency, opt-out, etc.)
7. User clicks "NEXT" button to submit form
8. Form validates all fields and consent
9. User receives immediate confirmation message on the page

### 3. How Users Provide Phone Numbers
- **Method:** Direct user input via web form
- **Field:** Clearly labeled "Phone Number" with required asterisk
- **Format:** US phone number format (XXX) XXX-XXXX
- **Validation:** Phone number is validated before submission
- **No Third-Party Collection:** Phone numbers are collected directly from users, not from purchased lists or third parties

### 4. When and Why Users Receive SMS
Users receive SMS messages in the following scenarios:

**Message Types:**
1. **Early Access Notifications**
   - Trigger: New exclusive opportunity becomes available
   - Example: "New exclusive opportunity available! Get early access at [link]"

2. **Promotional Offers**
   - Trigger: Special deals or discounts launch
   - Example: "VIP Exclusive: 20% off all products today only. Use code VIP20 at checkout."

3. **Product/Service Updates**
   - Trigger: New products or services are announced
   - Example: "New product launch! Be among the first to access [product name]."

4. **Transactional Messages**
   - Trigger: Account-related events
   - Example: "Your VIP account has been updated. View changes: [link]"

**Message Frequency:**
- Up to 10 messages per month
- Frequency may vary based on available opportunities
- Users are informed of this frequency before and after opt-in

### 5. How Users Can Opt-Out
Users can opt-out at any time through multiple methods:

**Via SMS Keywords (Immediate):**
- Reply **STOP** to any message
- Reply **UNSUBSCRIBE** to any message
- Reply **END** to any message
- Reply **QUIT** to any message
- Reply **CANCEL** to any message
- Reply **STOP ALL** to any message

**Via Customer Care:**
- Email: support@riosllc.com
- Phone: (201) 555-0123

**Opt-Out Confirmation:**
After sending an opt-out keyword, users receive:
"You have successfully been unsubscribed from RIOS LLC. You will not receive any more messages from this number."

**Opt-Out Process Documentation:**
- Documented in Terms of Service (Section 4)
- Documented in Privacy Policy (Section 7)
- Included in consent checkbox text
- Included in confirmation message
